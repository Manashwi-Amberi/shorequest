    About ShoreQuest

    ShoreQuest is a smart coastal travel and beach safety platform designed to help travelers explore beaches across India with greater confidence and awareness.

    The platform provides real-time weather conditions, wind speed analysis, and beach safety insights using live API data. Users can search and filter beaches based on safety conditions to plan safer and more enjoyable travel experiences.

    Built with modern web technologies, ShoreQuest aims to combine tourism, safety, and technology into one seamless experience for beach explorers, travelers, and coastal enthusiasts.
